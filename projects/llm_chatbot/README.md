# Ask ChatGPT python script

## What is this?
A lightweight Python script that allows users to interact with ChatGPT directly from the command line. No browser required..

By default, the script prompts the user for input. However, it can be easily modified to run automatically, enabling integration into larger workflows, scripts, or applications.

## Why is this interesting?
Today, many companies use large language models (LLMs) to automate a wide range of tasks, including:
Cleaning and formatting large volumes of documents

- Summarizing data
- Powering chatbots and virtual assistants
- Performing classification tasks
- Conducting sentiment analysis

This script can be adapted for any of the use cases above, making it a flexible starting point for automation and AI-powered tooling.

Important: Large language models can generate incorrect information ("hallucinations") and may reflect biases present in their training data. Always validate outputs and use responsibly.
